# keyboard

Link: https://keyboard-xd.netlify.app

UZ:
Bu sayt orqali siz klaviaturada qanday qilib to'gri yozish va klaviaturada barmoqlarni joylashishini o'rganishingiz mumkun. Sozlar JavaScriptning maxsus so'zlari generatsiyasi. Sayt har yangilanganida so'zlarni joylashuvi o'zgaradi.

ENG:
Through this website, you can learn how to type correctly and position your fingers on the keyboard. The words are generated by JavaScript's special words generation. The placement of words changes with each update of the website.

РУ:
Через этот сайт вы можете научиться правильно писать на клавиатуре и размещать пальцы на клавиатуре. Слова генерируются специальными терминами JavaScript. Расположение слов меняется при каждом обновлении сайта.
